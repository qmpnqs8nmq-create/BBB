## 20:47 修复 zenmux 402 不触发 key rotation 的 bug

### 根因
OpenClaw 的 `RAW_402_MARKER_RE` 正则不匹配裸 `402 {json}` 格式的错误文本。
Anthropic Messages API adapter 返回的原始错误是 `402 {"error":{...}}`（无 "HTTP" 前缀），
导致 `classifyFailoverReasonFrom402Text()` 返回 null，不触发 credential rotation。

### 修复
在 `/opt/homebrew/lib/node_modules/openclaw/dist/pi-embedded-CbCYZxIb.js` 的 `RAW_402_MARKER_RE` 末尾
添加 `|^\s*402\s*[\{\[]` 来匹配裸 402 + JSON 格式。同步修了 `LEADING_402_WRAPPER_RE`。
原文件已备份为 `.bak-402fix`。

### ⚠️ 注意
- 这是对 dist 的 monkey-patch，OpenClaw 更新后会被覆盖
- 待 Gateway restart 后生效
- 应向 OpenClaw 上游提 issue/PR
